# SCT_DS_03 – Interactive Visualization Dashboard

## 📌 Task Objective
Create an interactive dashboard using Tableau to analyze the Sample Superstore dataset and provide meaningful business insights through visualizations.

## 🛠️ Tools Used
- Tableau Desktop Free Edition
- Sample Superstore Dataset

## 📊 Dashboard Features
- Sales by Category (Bar Chart)
- Profit by Category (Bar Chart)
- Monthly Sales Trend (Line Chart)
- Interactive Category Filter

## 📈 Key Insights
- Technology category generated the highest sales.
- Office Supplies recorded the highest profit.
- Monthly sales showed fluctuations with noticeable peaks during certain months.
- Interactive filters allow users to explore category-wise performance.

## 📂 Files Included
- Task3_Interactive_Dashboard.twbx
- sample-superstore.xls
- Dashboard_Screenshot.png
- README.md

## 🎯 Outcome
Built an interactive Tableau dashboard that helps visualize sales performance, profit distribution, and monthly sales trends, enabling users to make data-driven business decisions.
